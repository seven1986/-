支付网关
=================
.. Note::

	订单创建、查询、退款和接收网关返回的支付通知等操作。支持的支付网关有：支付宝(Alipay)、微信支付(Wechatpay)、银联支付(Unionpay)、QQ钱包支付(Qpay)
